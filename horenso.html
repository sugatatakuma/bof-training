報連相って
なに？

報連相の重要性

仕事は "チームプレー"。だから、情報共有がすべての土台になります。
  • ✓
    どんな仕事も、必ず誰かと一緒に進めている
  • ✓
    上司やお客様は、あなたの状況を"見える化" してほしいと思っている
  • ✓
    正確でスピーディな共有が、トラブルを未然に防ぐ
報連相は、社内にも社外にも効く"最強のコミュニケーション術"です。
お客様の安心感・満足度アップにもつながります。

Section 02

報告・連絡・相談
を知る

報告とは

指示された仕事の結果や経過を、指示した人に伝えること。
報告
報告は"過去の事実"を共有すること。
上司やお客様から頼まれた仕事の進み具合、完了報告、トラブル報告などが該当します。
会議の議事録も報告の一種です。
👩
指示者（上司）
指示・命令 →
← 報告
👨
あなた

報告がないと・・・

上司にとって報告は、次の指示や判断を下すための"生命線"です。

それぞれどうなるか考えてみよう。クリックで答えが表示されます。

  • ✓
    進捗を報告していない👉 クリックで表示→ 上司が不安になり、わざわざ確認する手間が発生
  • ✓
    クレームを報告していない👉 クリックで表示→ 傷口が広がり、被害を最小限に抑えられない※報告しないうちは部下の責任、報告したら上司の責任
  • ✓
    変更点を報告していない👉 クリックで表示→ いざという時、上司がフォローできない

連絡とは

知った情報や変化を、関係者に正確に伝えること。
連絡
連絡は"変化があった情報"を共有すること。
日程変更、手順変更、ルール変更など、みんなに影響する情報をタイムリーに届けましょう。
報告と違い、上下関係に関わらず誰にでも行います。
👩
上司
← 連絡
👨
あなた
連絡 →
👩
同僚

連絡がないと・・・

連絡は、チームの情報を常に最新に保つための手段です。

それぞれどうなるか考えてみよう。クリックで答えが表示されます。

  • ✓
    打合せ時間を当日になって連絡した👉 クリックで表示→ 相手がタスクをコントロールできない
  • ✓
    遅刻するのに事前連絡をしていない👉 クリックで表示→ 不安にさせる、相手の時間を奪う
  • ✓
    プロジェクトの変更点を連絡していない👉 クリックで表示→ ズレが生じ、後々やり直しが発生

相談とは

困ったこと・迷ったことを経験者に話して、解決の糸口を見つけること。
相談
相談は、困ったことを解決するために助けを求めること。
ミスやクレーム、納期遅れなどのトラブルを上司に共有し、早期解決を目指します。
一人で抱え込まず、周りの力を借りましょう。

上司
← 相談

相談がないと・・・

上司にとって相談は、あなたをサポートするチャンスです。遠慮は不要。

それぞれどうなるか考えてみよう。クリックで答えが表示されます。

  • ✓
    分からないまま勝手に進めた👉 クリックで表示→ 方向違いで全部やり直し…
  • ✓
    一人で複数人のプロジェクトを進めた👉 クリックで表示→ 視点が偏り、クオリティが低下
  • ✓
    目標未達が見えていたのに相談しなかった👉 クリックで表示→ 上司の責任問題にまで発展

「仕事はできるのに報連相ができない」

📉

チーム全体の仕事が止まる

情報が共有されないと
チーム全体の動きが止まります

報連相は"できて当たり前"。できないとそれだけでマイナス評価になります。

お客様からいただいたネガティブな「声」

👤
依頼はしたものの、イメージと違う成果物が提出されたため、イメージとズレないように、もっと早めに相談してもらいたかった
👤
対応が遅いのは最悪許せるが、せめて進捗報告が欲しい
👤
レスは早いが、進捗がわからない時があり、「どのくらいかかります」とか連絡をくれると嬉しい
👤
しばらく未回答で「この件どうなっていますか？」と聞かないと状況がわからず、タスクを頭から離せない

報連相ができない理由 ── あるあるチェック

心当たりはありませんか？ クリックすると対処法が表示されます。

  • ✓ 「わざわざ言う必要ないかな」と自己判断してしまう +
    → あなたが "不要" と思っても、上司にとっては重要な情報かも。判断するのは受け手です！
  • ✓ タイミングを逃して後回しにしてしまう +
    → 遅れてでも伝える方が、伝えないよりずっと良い。「遅くなりましたが」の一言を添えましょう。

    → 悪い報告こそ早く！ 早めに伝えれば上司が一緒にリカバリーしてくれます。

    → 後回しにするほど言いづらくなります。「5分だけお時間いいですか？」の一言で大丈夫！

Section 04

報連相の
ルール

報告のルール

  • 1
    結論から話す。次に経過・状況・原因「結論は○○です。理由は～」の順番が鉄則
  • 2
    悪い報告こそ早く遅れるほど状況は悪化します。早めに伝えれば上司がフォローできます
  • 3
    事実と意見を区別する。まず事実（数値）→ 次に意見混ざっていると、受け手が正しい判断をできません
簡単な報告は3分以内が目安。要点を絞って簡潔に。

報告の6つのタイミング

「いつ報告すればいいの？」と迷ったら、この6つを思い出してください。

01
仕事が終わったとき
完了報告は基本中の基本
02
長期タスクの中間報告
進捗を定期的に共有
03
進め方に変更が必要なとき
方向転換は早めに相談
04
新しい情報を入手したとき
判断に影響する情報は即共有
05
改善方法を見つけたとき
良いアイデアも立派な報告
06
ミスをしたとき
隠さず、すぐに上司へ

連絡のルール

  • 1
    相手に伝わった時点で完了送っただけでは不十分。重要なメール・文書は「見ていただけましたか？」と確認しましょう
  • 2
    省略せず、抜け漏れなく伝える「これくらいわかるでしょ」は禁物。相手の立場で情報を整理して伝えましょう
  • 3
    内容だけでなく、伝える「人」の抜け漏れも確認「この情報、あの人にも必要かも？」と一度立ち止まって考えましょう

相談のルール

  • 1
    まず指示を出した人に相談する仕事の目的や欲しい結果が明確なので、一番的確なアドバイスがもらえます
  • 2
    相談前に内容を整理する：やったこと ＋ 困っていること ＋ 自分の案資料や現物を持っていくと伝わりやすい。案が浮かばない時は「案がないのですが」と正直に
  • 3
    ミス・トラブルは即相談早ければ早いほどダメージは小さくなります。一人で解決しようとしないで
Section 05

お客様の声
事例紹介

お客様からいただいたポジティブな「声」

👤
税務等の分からない事があれば会社に確認して報告してもらえるのでありがたい
👤
疑問・質問あれば、直ぐに連絡をくれるので、安心感がある
👤
タスク完了後に必ず報告をくれるので、安心感がある
👤
未消化に関して定期的に連絡をくれるので、業務依頼がしやすい
報連相の徹底が「安心感」「信頼」「また依頼したい」につながっています。

最上級の報連相：事例

「報連相」とセットで「スケジュールをコントロールすること」が重要。
お客様対応事例（BOFメンバー）
作業日前
  • 事前に作業スケジュールをお客様と擦合せ
  • 当日は質問があるかもしれないと事前連絡
作業当日
  • 不明点はためらわずに質問
  • 「重ねてのご質問失礼します」のクッション言葉
  • 作業終了時に実績工数と成果物を報告
ポイント！
  • ✓
    「作業スケジュール」を事前に握る
  • ✓
    質問をたくさんする可能性があることを事前に伝える
  • ✓
    クッション言葉で気遣いを見せる
  • ✓
    作業終了時に実績を必ず報告

最上級の報連相 ── 2時間22回のラリー

リモートでも「隣にいるんじゃないか？」という距離感を生んだチャットログ（抜粋）

担当者1:54 PM
本日15時頃から作業予定です。2時間ほど作業可能です。不明点あればご連絡します。 ▶ 作業開始予定＆予定工数の事前宣言
担当者3:16 PM
早々にご質問失礼いたします。○○様の勤務表ですが、有休を含めた161:43が稼働時間でよろしいでしょうか。 ▶ クッション言葉＋具体的な質問
担当者4:22 PM
自身の想定より進みが遅くなっております。 ▶ 遅延も正直に共有 → お客様「ゆっくりで大丈夫ですよー！」
担当者5:17 PM
本日は2時間半作業しまして、以下の格納分が完了しております。 ▶ 作業終了宣言＋実績工数＋成果物の報告
お客様8:30 PM
勤務表きれいにそろえられてる！ばっちりです。ありがとうございます！！

チャットログから学ぶ4つのポイント

カードをクリックすると答えが表示されます。

Q1. 作業前に何をした？
→ スケジュールを事前に共有し、質問する可能性があることも伝えた
Q2. 質問するときに心がけたことは？
→ クッション言葉を使い、曖昧なまま進めず必ず確認
Q3. 進みが遅いときどうした？
→ 正直に状況を共有。隠さず伝えたからお客様も安心できた
Q4. 作業終了時に何をした？
→ 実績工数＋成果物をセットで報告。次回の予定も共有
結果：2時間で22回のやり取り。リモートでも「隣にいるような安心感」を実現！

できていなければ不信感・不安感を与え評価が下がります。

「この報告(連絡)はこの人にもした方がいいのかな？」── 迷ったらGO！！
